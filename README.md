须弥（英文: Sumeru）。

> 须弥芥子 
> 
> 【解释】意思是偌大一个须弥山塞进一粒小小的菜籽之中刚刚合适。形容佛法无边，神通广大。也形容诗文波诡变幻，才思出众。
>
> 【出处】《维摩经不思议品》：“若菩萨住是解脱者，以须弥之高广，内芥子中，无所增减。”
